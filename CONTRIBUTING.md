# Contributing to rich-typography

## How can I contribute?

1. Report a bug
2. Add a new feature
3. Fix a bug
4. Improve the documentation

## Setup

1. Ensure uv is installed ([instructions][uv])
2. Clone and enter the rich-typography repository
3. Run `uv sync` to install the project dependencies
4. Activate the virtual environment `source .venv/bin/activate`

## Demo

```bash
python -m rich_typography
```

[uv]: https://docs.astral.sh/uv/getting-started/installation/
