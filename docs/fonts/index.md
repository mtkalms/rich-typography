# Fonts

## Regular

**Condensed Sans** (condensedsans)
```{.rich columns="120"}
from rich_typography import Typography
text = "The quick brown fox [underline]jumps[/underline] over the lazy dog"
output=Typography.from_markup(text, font="condensedsans")
```
