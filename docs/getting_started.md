# Getting started

## Requirements

Rich-typography requires Python 3.9 or later. Other than that, rich-typography runs anywhere Rich and Textual run.

## Installation

=== "uv"

    ```bash
    uv add rich-typography
    ```

=== "poetry"

    ```bash
    poetry add rich-typography
    ```

=== "pip"

    ```bash
    pip install rich-typography
    ```
